6. api/deleteVendorById/{id}---->delete vendor by id
7. api/saveDrug----------------->save the drug if the customer is available then
8. api/getAllDrug--------------->get all the data from drug module
9. api/getById/{id}------------->get the specific drug details by id
10.api/updateById/{id}---------->update the specific drug module by their id
11.api/deleteById/{id}---------->delete the specific drug by their id

13.api/getAllCustomer----------->get all the customer details
